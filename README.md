# HTML-TASKS

## Frontend Projects Repository

This repository contains multiple frontend projects built using **HTML, CSS, and JavaScript**. Each project focuses on a different UI/UX concept, ranging from a WhatsApp-style chat interface to a modern SaaS homepage. These projects are designed to demonstrate responsive design, interactivity, and modern web development techniques.

## Projects

### **Task 1: WhatsApp-Style Chat UI**
A responsive chat interface that mimics WhatsAppâ€™s look and feel.

**Features:**
- Left/right chat bubbles for user and bot
- Timestamps for every message
- Scrollable chat area
- Rounded input field and send button
- Simulated bot responses

**Technologies:** HTML, CSS, JavaScript

**Usage:**
1. Open `chatui/index.html` in your browser.
2. Type a message and press Enter or click Send.
3. Messages will appear with a simulated bot reply.

### **Task 2: Music Song Player**
A simple music player interface with interactive controls.

**Features:**
- Play, pause, next, previous song functionality
- Song list with track names
- Responsive player design
- Volume control and progress bar

**Technologies:** HTML, CSS, JavaScript, Audio API

**Usage:**
1. Open `music-player/index.html` in your browser.
2. Click on a song to play it.
3. Use the player controls to pause, skip, or adjust volume.

---

### **Task 3: Personal Portfolio Website**
A modern, responsive personal portfolio to showcase projects and skills.

**Features:**
- Home, About, Projects, and Contact sections
- Responsive layout for mobile and desktop
- Smooth scrolling and interactive navigation
- Project cards with descriptions and links
- Contact form (frontend only)

**Technologies:** HTML, CSS, JavaScript

**Usage:**
1. Open `portfolio/index.html` in your browser.
2. Navigate through the sections to explore the portfolio.

### **Task 4: Modern SaaS Homepage**
A responsive homepage design for a SaaS product.

**Features:**
- Hero section with call-to-action buttons
- Features section with icons and cards
- Testimonials section
- Pricing plans section
